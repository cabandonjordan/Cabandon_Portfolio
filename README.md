# 💻 SYSTEM_PROFILE // JORDAN A. CABANDON

Welcome to my interactive, OS-themed developer portfolio. Built with **React**, **Vite**, and **Three.js** using a futuristic cyber-aesthetic with a deep crimson and true black color palette.

## 🚀 Features

*   **Retro-Futuristic OS UI:** Interactive terminal windows, glitch text effects, and data-card structures.
*   **3D Interactive Background:** Uses an animated `Three.js` (React Three Fiber) Beams effect customized to a sleek dark red theme from ReactBits.
*   **Scroll Animations:** Smooth entry transitions powered by `AOS` (Animate On Scroll).
*   **Fully Responsive:** Designed to look crisp and functional on both mobile devices and large desktop monitors.
*   **Minimalist & Fast:** Lightweight pure CSS stylesheets with zero heavy CSS framework dependencies. 

## 🛠️ Technology Stack

*   **Frontend Framework:** React 19 + Vite
*   **3D Graphics / Background:** [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber), [@react-three/drei](https://github.com/pmndrs/drei)
*   **Animations:** [AOS](https://michalsnik.github.io/aos/) (Animate on Scroll)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Styling:** Pure Modern CSS (Custom variables, glassmorphism, CSS grids, flexbox)
*   **Version Control:** Git, GitHub

## ⚙️ Local Setup & Installation

To run this project locally, execute the following commands in your terminal:

1. **Clone the repository**
   ```bash
   git clone https://github.com/cabandonjordan/Cabandon_Portfolio.git
   cd Cabandon_Portfolio/cabandon-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   *The app will be available at `http://localhost:5173`*

4. **Build for production**
   ```bash
   npm run build
   ```
